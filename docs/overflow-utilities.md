Adjust the `overflow` property.

```html
<div class="-overflow-hidden">Overflow hidden</div>
```

__Optional classes:__

- `-overflow-hidden`
- `-overflow-visible`
- `-overflow-x-hidden`
- `-overflow-x-scroll`
- `-overflow-x-auto`
- `-overflow-y-hidden`
- `-overflow-y-scroll`
- `-overflow-y-auto`
