Adjust the `overflow` property.

```html
<div className="-overflow-hidden">Overflow hidden</div>
```

**Optional classes:**

- `-overflow-hidden`
- `-overflow-visible`
- `-overflow-x-hidden`
- `-overflow-x-scroll`
- `-overflow-x-auto`
- `-overflow-y-hidden`
- `-overflow-y-scroll`
- `-overflow-y-auto`
