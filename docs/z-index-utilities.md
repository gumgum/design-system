Adjust any element's z-index. Higher values render in front of lower values, `-z-neg` will render z-index: -1.

```html
<div className="-z-1">Z-Index 100</div>
```

**Optional classes:**

- `-z-1`
- `-z-2`
- `-z-3`
- `-z-4`
- `-z-5`
- `-z-6`
- `-z-7`
- `-z-8`
- `-z-9`
- `-z-neg`
