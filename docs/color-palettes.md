All colors used in the Design System are provided below, for use in JavaScript:

## Blue Theme

The `blue` theme is used on the following products:

- GumGum Dashboard
- GumGum Dental
- GumGum Dental Uploader (demo tool)

```javascript
const palette = {
    "blue": {
        "primary": "#00a7cf",
        "secondary": "#ffb819",
        "tertiary": "#00c4b4",
        "success": "#00c4b4",
        "info": "#555387",
        "warning": "#ffb819",
        "danger": "#ff5a34",
        "blue": "#00a7cf",
        "gold": "#ffb819",
        "green": "#00c4b4",
        "orange": "#ff6c0c",
        "purple": "#555387",
        "red": "#ff5a34",
        "darkBlue": "#0085a5",
        "darkGold": "#e5a516",
        "darkGreen": "#3fae2a",
        "darkRed": "#e5512e"
    }
};
```
