Adjust vertical alignment. Vertical alignment defaults to `baseline`.

```html
<div class="-va-middle"></div>
```

__Optional classes:__

- `-va-sub`
- `-va-super`
- `-va-top`
- `-va-middle`
- `-va-bottom`
- `-va-text-top`
- `-va-text-bottom`
