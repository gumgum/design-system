Adjust vertical alignment. Vertical alignment defaults to `baseline`.

```html
<div className="-va-middle"></div>
```

**Optional classes:**

-   `-va-sub`
-   `-va-super`
-   `-va-top`
-   `-va-middle`
-   `-va-bottom`
-   `-va-text-top`
-   `-va-text-bottom`
