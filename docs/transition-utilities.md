Adds transitions of varying lengths and easing values to any element.

```html
<p class="-transition-back-100">Transition on this div.</p>
```

__Optional classes:__

- `-transition-back-100`
- `-transition-back-250`
- `-transition-back-500`
- `-transition-out-100`
- `-transition-out-250`
- `-transition-out-500`
- `-transition-in-100`
- `-transition-in-250`
- `-transition-in-500`
