Adjust the float property on any element, and clear the floats wherever needed.

```html
<div className="-float-left">Float left</div>
<div className="-float-right">Float right</div>
<div className="-clear-both">Clear floats</div>
```

**Optional classes:**

- `-float-left`
- `-float-right`
- `-float-none`
- `-clear-left`
- `-clear-right`
- `-clear-both`
- `-float-none--xs`
- `-float-none--sm`
- `-float-none--md`
- `-float-none--lg`
- `-float-none--xl`
