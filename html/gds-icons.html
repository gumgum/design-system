Basic Icon Examples

flag

Black Tie Icons

You can place Black Tie icons just about anywhere using the proper CSS Prefix (bts, btb, btr, btl, or fab) followed by bt-iconname. 

<i class="btl bt-flag"></i> flag

fonticons

Font Awesome Brand Icons

You can place Black Tie icons just about anywhere using the CSS Prefix fab followed by fab-iconname. 

<i class="fab fab-fonticons"></i> fonticons

Black Tie and Font Awesome Brand icons are designed to be used with inline elements (we like the <i> tag for brevity, but using a <span> is arguably more semantically correct).

  • If you change the font-size of the icon's container, the icon gets bigger. Same things goes for color, drop shadow, and anything else that gets inherited using CSS.

Icon Size Examples

  bt-sm

  bt-lg

  bt-2x

  bt-3x

  bt-4x

  bt-5x

To change icon sizes relative to their container, use the bt-sm (25% decrease), bt-lg (33% increase), bt-2x, bt-3x, bt-4x, or bt-5x classes. 

<i class="btl bt-flag bt-sm"></i> bt-sm
<i class="btl bt-flag bt-lg"></i> bt-lg
<i class="btl bt-flag bt-2x"></i> bt-2x
<i class="btl bt-flag bt-3x"></i> bt-3x
<i class="btl bt-flag bt-4x"></i> bt-4x
<i class="btl bt-flag bt-5x"></i> bt-5x
  • If your icons are getting chopped off on top and bottom, make sure you have sufficient line-height.

Fixed Width Icon Examples

  Home
  Library
  Applications
  Settings

Use bt-fw to set icons at a fixed width. Great to use when different icon widths throw off alignment. Especially useful in things like nav lists & list groups. 

<div class="list-group">
  <div class="list-group-item"><i class="btl bt-home bt-fw"></i> Home</div>
  <div class="list-group-item"><i class="btl bt-book bt-fw"></i> Library</div>
  <div class="list-group-item"><i class="btl bt-pencil bt-fw"></i> Applications</div>
  <div class="list-group-item"><i class="btl bt-gear bt-fw"></i> Settings</div>
</div>

Bordered & Pulled Icon Examples

…tomorrow we will run faster, stretch out our arms farther… And then one fine morning So we beat on, boats against the current, borne back ceaselessly into the past.

Use bt-border and pull-right or pull-left for easy pull quotes or article icons. 

<i class="btl bt-quote-left bt-3x pull-left bt-border"></i>
...tomorrow we will run faster, stretch out our arms farther...
And then one fine morning So we beat on, boats against the
current, borne back ceaselessly into the past.

Animated Icon Examples

Use the bt-spin class to get any icon to rotate, and use bt-pulse to have it rotate with 8 steps. Works well with bt-spinner, bt-spinner-clock, bt-sync, and bt-gear. 

<i class="btl bt-spinner bt-spin"></i>
<i class="btl bt-spinner-clock bt-spin"></i>
<i class="btl bt-sync bt-spin"></i>
<i class="btl bt-gear bt-spin"></i>
<i class="btl bt-spinner bt-pulse"></i>
  • CSS3 animations aren't supported in IE8 - IE9.

<i class="btl bt-shield bt-rotate-180"></i> bt-rotate-180<br>

<i class="btl bt-shield bt-flip-horizontal"></i> bt-flip-horizontal<br>
<i class="btl bt-shield bt-flip-vertical"></i> bt-flip-vertical

To stack multiple icons, use the bt-stack class on the parent, the bt-stack-sm for the small icon, and bt-stack-1x for the regular sized icon. bt-inverse can be used as an alternative icon color. You can even throw larger icon classes on the parent to get further control of sizing. 

<span class="bt-stack bt-2x">
  <i class="btl bt-square bt-stack-1x"></i>
  <i class="fab fab-twitter bt-stack-sm"></i>
</span>
